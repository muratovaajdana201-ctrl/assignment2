# Assignment 2

## Реализованные структуры данных
- MyArrayList
- MyLinkedList
- MyStack
- MyQueue
- MyMinHeap

## Описание
В этом проекте реализованы собственные структуры данных на Java без использования стандартных коллекций из java.util.

## Файлы
- MyList.java — интерфейс
- MyArrayList.java — динамический массив
- MyLinkedList.java — двусвязный список
- MyStack.java — стек
- MyQueue.java — очередь
- MyMinHeap.java — минимальная куча
- Main.java — тестирование

## Особенности
- Используются generics
- Обработка ошибок
- Все структуры протестированы
